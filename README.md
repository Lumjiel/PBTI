# PBTI - 乒乓球人格测试

🏓 30道题，测出你的乒乓人格。

## ✨ 功能特性

- **人格测试** - 30道趣味选择题，发现你的乒乓人格类型
- **五大维度** - 燃度、韧度、精度、可见度、黏度
- **10+1人格** - 匠人、暴力熊猫、卡尺、年轮等10种人格 + 隐藏香蕉人格
- **雷达图展示** - 可视化呈现你的性格光谱
- **海报分享** - 生成专属测试结果海报，一键分享
- **二维码分享** - 扫码即可查看测试结果

## 🛠️ 技术栈

- **框架** - [Next.js 15](https://nextjs.org/)
- **UI库** - [React 19](https://react.dev/)
- **语言** - [TypeScript 5](https://www.typescriptlang.org/)
- **样式** - [Tailwind CSS 4](https://tailwindcss.com/)
- **图标** - 内置 CharacterSVG 组件

### 人格简介

- **匠人 (FORGE)** - 一个动作一万遍，重复即修行
- **暴力熊猫 (FUSE)** - 战术？不用。纯粹质量碾压
- **卡尺 (CLOCK)** - 对精准有病态执念
- **马大厨 (MAZE)** - 算计到毫厘，摆短劈长发球变化无穷
- **帝国之壁 (WALL)** - 不主动出击，但谁也过不去
- **归因怪 (ARCHIVE)** - 必须知道为什么输，答案比结果更重要
- **回声 (ECHO)** - 不用从头来过的感觉，真好
- **攒局怪 (HUB)** - 玩什么不重要，重要的是和谁玩
- **勿扰 (ISLE)** - 独处是充电，人群是消耗
- **仓鼠 (VAULT)** - 收集的不是物件，是时间胶囊
- **苦行 (MONK)** - 每一步都算数，慢即是快
- **演员 (STAGE)** - 赢不赢不重要，姿势要帅
- **冤魂 (GHOST)** - 输在起跑线，但绝不会缺席
- **梁满局 (OAK)** - 年轮一圈圈，球路一步步
- **水仙 (MIRROR)** - 我即风格，风格即我
- **新风 (WING)** - 轻盈到让对手感觉在和影子打球
- **球场做题家 (DAWN)** - 每一球都是卷子，每一分都是答案
- **吉吉国王 (STORM)** - 风风火火，惊惊乍乍
- **随缘 (LEAF)** - 球来伸手，该有的都会有
- **天鹅 (SWAN)** - 表面波澜不惊，暗里翻江倒海
- **云佬 (TOME)** - 理论满分，实操是另一个部门的事
- **泥洼老师 (ZEN)** - 看起来啥都没想，实际上在好几层以下
- **野火 (WILD)** - 没学过规矩，我的风格是自己长出来的
- **帝国の绝凶虎 (RISE)** - 顺风局昏睡，逆风局觉醒
- **香蕉 (BANANA)** - 你以为你在第五层，其实你在水果店

## 📝 五大维度

每个维度分为 **L1-L6** 六个等级：

### 🔥 燃度 (BURN)

| 等级 | 描述 |
|------|------|
| L1 | 省电模式，能不动就不动 |
| L2 | 文火慢炖，持久输出 |
| L3 | 偶有高潮，但能收回 |
| L4 | 稳定输出，偶有爆点 |
| L5 | 高能预警，频繁燃爆 |
| L6 | 全功率永动机，停不下来 |

### 💪 韧度 (TOUGH)

| 等级 | 描述 |
|------|------|
| L1 | 风吹即散，一压就垮 |
| L2 | 小逆还行，大逆摆烂 |
| L3 | 能撑一会儿，但会崩 |
| L4 | 有韧性，但会消耗自己 |
| L5 | 越压越强，绝境逆袭 |
| L6 | 逆境剧本狂，落后才来劲 |

### 🎯 精度 (PREC)

| 等级 | 描述 |
|------|------|
| L1 | 大而化之，差不多就行 |
| L2 | 过得去就行，别太讲究 |
| L3 | 重要的事会认真 |
| L4 | 有追求，但能妥协 |
| L5 | 细节控，不达标准不舒服 |
| L6 | 显微镜成精，差一点也不行 |

### ✨ 可见度 (SHOW)

| 等级 | 描述 |
|------|------|
| L1 | 只想隐身，不想被看到 |
| L2 | 低调行事，被关注会慌 |
| L3 | 小圈子可以，大舞台社恐 |
| L4 | 有时想表现，有时想躲 |
| L5 | 喜欢被关注，表现欲强 |
| L6 | 观众是我的氧气，舞台是我的家 |

### 🤝 黏度 (BOND)

| 等级 | 描述 |
|------|------|
| L1 | 孤狼结界，一米距离 |
| L2 | 君子之交，淡如水 |
| L3 | 需要独处充电，但能社交 |
| L4 | 有来有往，但不主动 |
| L5 | 享受连接，主动社交 |
| L6 | 人群充电站，社牛本牛 |
